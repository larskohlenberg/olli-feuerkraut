# BBQ Brew Crew als Dachmarke

Die Website wird zur Hauptwebsite von BBQ Brew Crew. Feuerkraut bleibt erhalten, wird aber als Gewuerz-Produktlinie von BBQ Brew Crew gefuehrt, weil die offizielle Domain, das bestehende Catering-Angebot, Promotion, Grillteam, Kontakt und Impressum unter BBQ Brew Crew gebuendelt werden sollen.

## Considered Options

- Feuerkraut bleibt Hauptmarke: passend zur bisherigen Landingpage, aber widerspruechlich zur geplanten Ersetzung der offiziellen BBQ-Brew-Crew-Website.
- BBQ Brew Crew wird Dachmarke: passend zur offiziellen Website-Struktur und zum breiteren Leistungsangebot; Feuerkraut kann darunter klar als Produktlinie positioniert werden.

## Consequences

Hero, Navigation, SEO, Kontakt, Impressum und Leistungsstruktur fuehren BBQ Brew Crew als Marke. Feuerkraut-Texte muessen als Produktlinien-Texte formuliert werden und duerfen Catering, Promotion oder die gesamte Website nicht als Feuerkraut-Angebot rahmen.
