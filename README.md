# Olli Feuerkraut Landingpage

Branding- und Landingpage-Prototyp fuer die geplante BBQ-Gewuerzlinie von Oliver / BBQ Brew Crew in Zusammenarbeit mit dem Kraeuterpark Altenau.

## Inhalt

- `landing/index.html` - statische Landingpage.
- `landing/styles.css` - Styling und Responsive Layout.
- `landing/assets/` - generierte Produktmotive und kuratierte Arbeitskopien der Eventbilder.
- `docs/brandvoice-feuerkraut.md` - dokumentierte Brandvoice, Positionierung, Produktlogik und naechste Schritte.

## Lokale Vorschau

Die Seite kann direkt im Browser geoeffnet werden:

```text
landing/index.html
```

Alternativ mit einem lokalen Server:

```bash
cd landing
python3 -m http.server 8123
```

Dann `http://127.0.0.1:8123` oeffnen.

## Brand-Kern

> Vom Feuer getestet. Im Kraeuterpark gemischt.

Die Marke verbindet echte BBQ-Crew-Praxis, Kraeuterpark-Herkunft, Harz-Terroir und klare Anwendung am Grill.
